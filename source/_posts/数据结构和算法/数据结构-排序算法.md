冒泡排序：

依次比较相邻的两个数，将大的放到后面，这样第一轮排序后，最大的在最后面；如图：

![1.png](1.png)

代码实现：

```
    private static int[] bubbleSort(int[] sourceArray) {
        //拷贝，为了不改变参数内容
        int[] arr = Arrays.copyOf(sourceArray, sourceArray.length);

        //注意i=1
        for (int i = 1; i < arr.length; i++) {
            for (int j = 0; j < arr.length - i; j++) {
                if (arr[j] > arr[j + 1]) {
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                }
            }
        }

        return arr;

    }
```
